# O que o Open Data Londrina? 

Comunidade sem fins lucrativos com foco em dados, governança de dados, IA e ser um hub de dados da cidade de Londrina.

# Nossa Missão? 

Promover discussões sobre temas relacionados a dados — big data, data analytics, data engineering data science — inteligência artificial, governança e transparência de dados na esfera pública, proteção de dados, entre outros. Acreditamos que, por meio dessas discussões, seja possível fomentar o compartilhamento de conhecimento entre os membros da comunidade.

Além disso, com os datasets abertos e disponíveis, acreditamos que isso possa incentivar a criação de produtos que impactem positivamente a sociedade londrinense.
