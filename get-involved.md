# Como se envolver e contribuir com a comunidade Open Data Londrina

## Formas de se envolver e contribuir

### Dê uma Estrela

A forma mais simples de apoiar o projeto é dando uma estrela no repositório!

### Reporte Bugs ou Problemas

Econtrou algo errado? 

- Abra uma Issue com a tag 
- Descreva o problema de forma clara e objetiva
- Inclua examplos se possível
- Quer corrigir? Abra a Issue e submeta um Pull Request!

### Sugira Melhorias

Tem ideias para a comunidade, datasets ou outras sugestões? 
- Abra uma Issue com a tag improvement
- Explique a proposta de forma clara e objetiva
- Descreva porquê deve ser priorizada

### Contribua com código/conteúdo/eventos

- Crie novos datasets
- Sugira eventos
- Desenvolva novos produtos
- Melhores os produtos da comunidade

### Compartilhe conhecimento
- Participe de discussões nas Issues
- Compartilhe produtos que fez utilizando datasets da comunidade
- Marque a comunidade nas redes sociais
